###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greeting = {};
console.log(greeting);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `{}`

<i>Variables declared with the "let" keyword in JavaScript are `reassignable`. This means that you can change their value after the initial declaration.
When you declare a "greeting" variable using the "let" keyword without any value, its value is undefined. After reassigning the variable's "greeting" value to {}.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: - C: `"12"`

<i>If you concate a number to a string in JavaScript, the number is converted to a string. Here in the "sum" function, the first parameter is the number and the second is the string, the two are joined to become a string, that's why the answer is "12".</i>

</p>
</details>

###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
const food = ["🍕", "🍫", "🥑", "🍔"];
const info = { favoriteFood: food[0] };

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: `['🍕', '🍫', '🥑', '🍔']`

<i>The property "favoriteFood" in the Object "info" is assigned the value of the first element of the "food" array, which is "🍕".

Now, when a new value "🍝" is assigned to "info.favoriteFood", the value of the "favoriteFood" property in the "info" object is changed, but the direct "food" array is not changed. That's why the answer is "A".</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: `Hi there, undefined`

<i>When call the "sayHi" function without passing any arguments, the parameter name inside the function doesn't receive a value, so it defaults to "undefined". that's why the answer is "B".</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>Here the value of "count" is 0 by default. While "for.each" loops the nums array every time it gets a truthy value the count will be incremented by 1, the first value of the nums array is 0(falsy) and the remaining 3 values are truthy. So the value of "count" will be 3.</i>

</p>
</details>
